Lista de Tarefas

Seja bem-vindo à Lista de Tarefas!

Instruções de uso:

  1. Escreva a tarefa no campo abaixo;
  2. Clique no botão para adicionar à lista;
  3. Quando tiver concluído uma tarefa, clique na corresponde para riscá-la da lista;
  4. Caso tenha riscado a tarefa, clique nela duas vezes para desmarcar a tarefa.
